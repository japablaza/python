# Functions during my journey

## Built-in Functions

- bin(x)   
  x must be an integer number. bin() function will convert to a binary *string* prefixed with `0b`.  
  
   - Example and Format:  
     ```
     >>> bin(11)  
     '0b1011'
     ```

     ```
     >>>format(33, '#b')
     '0b100001'
     ```

     ```
     >>>format(21, 'b')
     '10101'
     ```

     ```
     >>>f'{55:#b}'
     '0b110111'
     ```

     ```
     >>>f'{78:b}'
     '1001110'
